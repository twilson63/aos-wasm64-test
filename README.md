# AOS Wasm64 Test

This repository is designed to test the compute performance of wasm64.

## Requirements

- NodeJS v22+ [Download NodeJS](https://nodejs.org)
- Yarn: Install with `npm install -g yarn`

## Usage

```sh
npx -y degit twilson63/aos-wasm63-test compute
cd compute
yarn
yarn start
```
